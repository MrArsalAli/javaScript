Increment/Decrement Exercises:

1. Create a variable count with an initial value of 8. Use the post-increment operator to increment count, then log the result to the console.

2. Initialize a variable x with a value of 15. Use the pre-decrement operator to decrement x, then display the updated value.

3. Declare a variable num and set it to 25. Use the post-decrement operator to decrement num, then print the result.

4. Start with a variable value set to 10. Apply the pre-increment operator to value, and then output the new value.

5. Set a variable index to 5. Apply the post-increment operator to index and display its value.

6. Define a variable quantity and assign it the value 12. Use the pre-decrement operator on quantity, and then show its updated value.

7. Initialize a variable counter with a value of 18. Use the post-increment operator on counter and log the result.

8. Create a variable score set to 30. Use the post-decrement operator on score, then print the result.

9. Start with a variable value set to 20. Apply the pre-increment operator to value, and then output the new value.

10. Declare a variable counter and set it to 8. Apply the post-decrement operator to counter, then display the result.

Concatenation Exercises:

1. Create a variable firstName and assign it the value 'John'. Then, concatenate it with the string 'Doe' and display the full name.

2. Declare a variable city and set it to 'New'. Next, concatenate it with the string 'York' and display the complete city name.

3. Define a variable fruit with the value 'apple'. Concatenate it with the string 'pie' to form a dessert, and then print it.

4. Initialize a variable animal with 'black'. Concatenate it with the string 'bird' and display the full animal name.

5. Combine the strings 'JavaScript' and 'is fun' to create a sentence, then display it.

6. Concatenate 'Open' with 'Source' and 'Intelligence' to form a technology company's name and show it.

7. Join the strings 'Happy', 'Birthday', and 'to you' to wish someone on their special day, and then display the greeting.

8. Concatenate 'Good' with 'Luck' and 'Have' with 'Fun' to create well wishes, and then display both phrases.

9. Combine the strings 'Web', 'Design', and 'Development' to describe a comprehensive skill set, and then print it.

10. Join 'Artificial', 'Neural', and 'Networks' to form a complex technology term, and then display it.
